This project creates an interactive environment where multiple AI models communicate with each other through natural language conversations.
The system integrates three powerful language models—GPT, Ollama, and Gemini—to simulate dynamic, multi-agent dialogue.
The goal is to explore how different AI models interpret, respond, and collaborate in a shared conversational space,
highlighting differences in reasoning, tone, and response generation.

System Architecture:
User Input
    ↓
Conversation Manager
    ↓
---------------------------------
|   GPT   |  Ollama  |  Gemini  |
---------------------------------
    ↓
Response Aggregation
    ↓
Output (Text / Speech)


🧰 Technologies Used:
Python 
OpenAI GPT API
Ollama API / Local Models
Google Gemini API
